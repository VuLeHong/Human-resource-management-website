Sau khi pull repo này về, hãy mở terminal và nhập các lệnh sau đây (nên tách ra làm 2 terminal độc lập để dễ dàng thao tác):
    cd Front_end/
    npm i
    npm i axios
    npm i react-router-dom
    npm i react-icons --save
    npm i recharts
    npm i --save chart.js react-chartjs-2
    npm run dev
    cd Back_end/
    npm i dotenv
    npm i express-async-handler
    npm start
